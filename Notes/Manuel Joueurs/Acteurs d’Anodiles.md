| Acteur                    | Complément d’information                                                                                                                                     |     |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ | --- |
| Pirate                    | Vit entre les routes officielles et les zones grises. Toujours toléré par quelqu’un… et recherché par un autre. Dépend des ports qui acceptent de l’oublier. |     |
| Explorateur               | Travaille rarement pour lui‑même. Ses découvertes appartiennent presque toujours à une faction, une académie ou un commanditaire qu’il craint.               |     |
| Agent impérial            | Porte l’Empire avec lui, même quand il agit “off the record”. Sa présence transforme tout incident local en affaire politique.                               |     |
| Coursier                  | Transporte informations, contrats et secrets plus souvent que des lettres. En sait toujours trop pour rester neutre longtemps.                               |     |
| Artisan                   | Maîtrise une technique rare (navale, magique, nécromantique, élémentaire). Ce qu’il produit crée des dépendances.                                            |     |
| Scientifique              | Observe, mesure et archive un monde instable. Ses travaux sont souvent récupérés, volontairement ou non, par une faction.                                    |     |
| Arcaniste                 | Manipule les flux élémentaires ou magiques sans jamais les contrôler totalement. Chaque usage laisse une trace.                                              |     |
| Mercenaire                | Vend sa compétence, jamais son allégeance officielle. Traîne une réputation qui le précède toujours.                                                         |     |
| Renégat                   | A rompu avec une faction, une loi ou une nature. Son passé revient toujours sous forme de chasse, de dette ou de rumeur.                                     |     |
| Capitaine de clan         | Responsable de vies, de routes et de réputation. Chaque décision sauve un groupe et en condamne un autre.                                                    |     |
| Cartographe               | Ne dessine pas seulement des cartes : il fixe le réel. Ses erreurs – volontaires ou non – coûtent des navires.                                               |     |
| Nourrisseur               | Gère la survie alimentaire. Utilise des solutions rapides que d’autres devront payer plus tard.                                                              |     |
| Épicier de la Confrérie   | Défend une vision à long terme du monde. Refuse certaines missions, même vitales à court terme.                                                              |     |
| Nécromancien              | Administre la mort comme une ressource. Son travail est légal, mais rarement accepté socialement.                                                            |     |
| Démoniste                 | Fonctionnaire avant d’être mystique. Cherche l’efficacité, tolère l’échec, normalise la perte humaine.                                                       |     |
| Marin pêcheur             | Dépend directement des humeurs du monde (météo, élémentaires, bêtes). Les premiers témoins des anomalies.                                                    |     |
| Balladin                  | Transporte récits, rumeurs et versions biaisées des événements. Façonne l’opinion plus sûrement qu’un décret.                                                |     |
| Veilleur orque            | Gardien d’un équilibre brutal. Protège le groupe au détriment de l’individu.                                                                                 |     |
| Adepte de la Pierre Verte | Voit dans la contamination une vérité cachée. Considère le monde comme déjà irrécupérable.                                                                   |     |
| Contremaître impérial     | Fait fonctionner la machine locale. N’a pas le pouvoir, mais applique l’ordre sans l’adoucir.                                                                |     |
| Réfugié des Vagues        | Survivant d’un désastre passé. Personnage mobile, méfiant, porteur de traumatismes utilisables en jeu.                                                       |     |
| Chasseur de monstres      | Spécialisé dans des menaces “locales”, mais chaque victoire révèle un problème plus vaste.                                                                   |     |
| Exploitant de ruines      | Pille des vestiges anciens, souvent sans comprendre ce qu’il déclenche.                                                                                      |     |

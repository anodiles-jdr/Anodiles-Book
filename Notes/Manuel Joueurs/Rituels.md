## 🔮 ARCANE

### Couleur masquée

Action : Create an Advantage  
Effet : Crée l’Aspect « Couleur elfique indétectable » sur une personne.  
Résiste aux tentatives d’identification magique ou sociale.

---

### Cargaison masquée

Action : Create an Advantage  
Effet : Le navire gagne « Soute illisible ».  
Toute inspection doit d’abord surmonter cet Aspect.

---

### Vision faussée

Action : Create an Advantage  
Effet : Le navire obtient « Apparence d’un autre navire ».  
Fonctionne tant que la fiction reste crédible.

---

### Couleur falsifiée

Action : Overcome ou Create an Advantage  
Effet : Altère temporairement la couleur elfique.  
Peut créer « Couleur d’emprunt instable » en cas d’échec.

---

### Drapeau magique

Action : Create an Advantage  
Effet : Le drapeau porte « Message arcanique scellé ».  
Lecture possible uniquement via Occultisme.

---

### Détection des rituels stockés

Action : Overcome  
Effet : Révèle les Aspects de rituel présents sur un objet ou un navire.

---

### Dé/chargement rapide

Action : Create an Advantage + Overcome  
Effet : « Manutention autonome » permet de charger/décharger rapidement une cargaison.

---

### Scan de bateau

Action : Overcome  
Effet : Le MJ doit répondre honnêtement à plusieurs questions sur le navire ciblé  
(origine, état, enchantements, danger).

---

### Rameurs fantômes

Action : Create an Advantage  
Effet : Le navire gagne « Propulsion spectrale ».  
Justifie vitesse accrue ou manœuvres impossibles sans équipage.

---

### Pièce rapetissante

Action : Create an Advantage  
Effet : La zone affectée gagne « Volume compressé ».  
Tout ce qui entre est réduit à des fins de stockage.

---

### Invisibilité de navire

Action : Create an Advantage (rituel risqué)  
Effet : « Navire hors-vue ».  
Les observateurs doivent surmonter cet Aspect pour détecter le navire.

---

### Stockage d’élémentaire

Action : Create an Advantage  
Effet : L’objet reçoit « Chargé d’élémentaire (X) ».  
Chaque charge peut être invoquée pour un effet lié à l’élément.

---
